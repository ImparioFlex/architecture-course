# Architecture Course

Standalone click-through HTML course for learning software architecture and AI systems conceptually.

## Open It

Open `/Users/trevor/Projects/CLAUDE/architecture-course/index.html` in a browser.

## What It Includes

- layered lesson depth (`Level 1`, `Level 2`, `Level 3`)
- hover glossary terms with quick definitions
- pinned glossary detail panel
- AI sidebar in every lesson
- Mermaid syntax snapshots for each concept
- simple diagrams and quizzes

## Expand It

All lesson content lives in `/Users/trevor/Projects/CLAUDE/architecture-course/script.js`.
Add new glossary entries to `glossary` and new lessons to `courses`.
